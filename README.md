# Landing Page — тестовое задание

**[Live Demo](https://artkruzhkov.github.io/testTask_SA/)**

Данный репозиторий содержит решение тестового задания.

Был взят готовый лендинг и выполнены следующие доработки согласно ТЗ:

## Что было сделано

- Полностью скопирован лендинг (HTML / CSS / JS / изображения).
- Удалены сторонние трекеры, аналитика и внешние скрипты.
- Все ресурсы переведены на локальные файлы.
- Настроен переход с лендинга на указанный URL (`https://google.com`).
- Реализована логика вращения колеса:
  - 1-й спин — проигрыш (грустный смайлик).
  - 2-й спин — выигрыш `$1000`.
  - 3-й спин — финальный выигрыш `$5000`.
- Изменены и добавлены попапы согласно требованиям задания.
- Удалён таймер из стартового попапа.
- Изменён текст и стили попапов.
- Добавлена финальная модалка с кнопкой перехода.
